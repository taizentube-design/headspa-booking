# 店舗データ（stores.json）

- **district** … カテゴリ（渋谷・新宿・銀座など）。東京一覧で「〇〇のヘッドスパ」としてグループ表示する。
- **hpb_url** … ホットペッパービューティー（第1優先・PR表記）
- **official_url** … 公式サイト（第2優先）
- **inquiry_url** … 問い合わせ（第3優先）
- **image_url** … 店舗画像URL（任意）。ある場合のみ一覧・詳細で表示。HPBの画像を使う場合は、サロンページを開き「ページのソースを表示」で `og:image` の `content="..."` のURLをコピーして貼る。引用表記は店舗説明の citation で行っている。

アフィリエイト用リンクを取得したら `hpb_url` を差し替えてください。新規店舗は `district` を入れると一覧でそのカテゴリに表示されます。
