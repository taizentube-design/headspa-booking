# 店舗データ（stores.json）

## 基本項目
- **district** … カテゴリ（渋谷・新宿・銀座など）。東京一覧で「〇〇のヘッドスパ」としてグループ表示する。
- **hpb_url** … ホットペッパービューティー（第1優先・PR表記）
- **official_url** … 公式サイト（第2優先）
- **inquiry_url** … 問い合わせ（第3優先）
- **image_url** … 店舗画像URL（任意）。ある場合のみ一覧・詳細で表示。HPBの画像を使う場合は、サロンページを開き「ページのソースを表示」で `og:image` の `content="..."` のURLをコピーして貼る。引用表記は店舗説明の citation で行っている。

## アクセス・営業（SEO・利便性）
- **nearest_station** … 最寄り駅（例：渋谷駅）。複数ある場合はメイン1つ。
- **walk_minutes** … 最寄り駅からの徒歩分数。数値（例：2）または文字（例：`1〜2`）可。
- **access_note** … アクセス補足（ビル名・階・目印）。例：`山口ビル2F`、`〇〇の隣`。
- **hours** … 営業時間（既存）。例：`11:00〜20:00`、`要確認`。
- **hours_note** … 営業時間の補足。例：`不定休あり`、`土日祝も営業`。
- **closed** … 定休日。例：`水曜`、`日祝`。空欄可。

## 特徴・料金
- **price_note** … 価格の目安（1行）。例：`クイックヘッドスパ 1,500円〜`、`リラクゼーションヘッドスパ 5,280円〜`。
- **features** … 特徴タグの配列。例：`["完全個室","深夜営業","24時間WEB予約可","学割U24","妊婦OK","メンズ専門"]`。空配列 `[]` 可。

## その他
- **payment** … 支払い方法の補足。例：`カード可、PayPay可`。空欄可。
- **parking** … 駐車場の有無・補足。例：`あり（2台）`、`近隣にコインパーキング`。空欄可。

---

アフィリエイト用リンクを取得したら `hpb_url` を差し替えてください。新規店舗は `district` を入れると一覧でそのカテゴリに表示されます。
